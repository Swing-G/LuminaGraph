# 试用升级引导

## 适用范围
试用客户咨询升级方案、功能对比、价格、到期影响等。

## SOP
1. 查询账号和订阅状态 — 确认试用期到期时间
2. 查询客户使用情况 — 团队规模、活跃度等（从账号信息获取）
3. 根据客户等级推荐合适的升级方案
4. 如有工单，确认之前是否有升级咨询记录

## 领域规则
- 试用到期前7天提醒 → 主动推送升级方案
- 试用到期后7天内 → 宽限期，数据保留
- 试用到期超过7天 → 数据可能被清理
- 推荐策略: 小团队→团队基础版 / 中型→企业专业版 / 大型→企业旗舰版
- 试用转正式享首月8折优惠

## 提示词模板
客户 {accountId} 处于试用期，咨询升级方案。请介绍当前试用版到期时间和功能限制，推荐合适的付费版本。

## 输出规范
- 先确认当前试用到期时间
- 列出各版本核心差异和价格
- 根据客户规模给出具体推荐
- 告知升级流程和优惠信息
